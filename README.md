## Installation

To get a local copy up and running, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/UzairAlii/MetroBank.git
    ```

2. Navigate to the project directory:
    ```bash
    cd MetroBank
    ```

3. Install the dependencies:
    ```bash
    npm install
    ```
